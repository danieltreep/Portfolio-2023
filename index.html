PROJECTS

RealEstateCare

VIEW

A webapplication made with Vue 3, VueX and VueRouter. I used Ionic as UI library to target the mobile feel. The purpose of the app is that home inspectors can fill in an inspection form on location. The form then gets saved in state to simulate a database. The stored inspections can be sorted on date, read, updated and deleted. Use the developer tools to simulate a mobile device if you're on desktop.

Member Administration

VIDEO

Using the MVC design pattern I created a member administration webapplication with PHP and MYSQL for the database. The site is not uploaded online but the source code can be viewed and there is a demo video. In the app it is possible to add members to a family and age group. Based on this the amount of contribution is automatically calculated. The members, families, age groups and amounts can all be updated using the CRUD methods.

Superbowl Predictions

VIEW

A small app I made for friends to predict the Superbowl. The app is made with React and React Router. Firstly choose who's predicting. Then predict per round who's winning by clicking the logos. The winners are added to an array and sorted to take the seeding in account. At the end the prediction will be nicely displayed with all the logos.
